# The Rental Hub

A web-based car rental platform developed to manage vehicle listings, pricing, user bookings, and admin operations through a user-friendly interface.

## Features
- Car listings
- Vehicle details page
- Search functionality
- Booking management
- User profile system
- Admin dashboard

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## Purpose
Developed as an academic project to simplify car rental browsing and management.
